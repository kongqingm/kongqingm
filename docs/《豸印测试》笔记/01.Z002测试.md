## 1、章筒绑定功能

```
(1)预置条件
	①小程序账号及公司已申请、登录
(2)操作步骤
    ①进入小程序【工作台】
    ②点击【添加印章】
    ③搜索连接印章，并输入印章名称
    ④点击【确认】
    ⑤进入【工作台】【印章列表】查看所绑定的章筒
(3)预期结果
    ①印章连接时秘钥验证成功
    ②印章被正确绑定
    ③小程序列表中显示该印章
    ④章筒状态变为已绑定状态，液晶屏上名称显示正确
```



## 2、章筒修改名称

```
(1)预置条件
    ①所要操作的章筒已被绑定
(2)操作步骤
    ①进入【通讯录】【印章列表】
    ②长按所要修改名称的章筒，选择【修改名称】
    ③输入所要修改的名称
    ④连接印章
    ⑤查看修改后，章筒、小程序显示的名称
(3)预期结果
    ①章筒小程序显示的章筒名称，正确被修改
    ②章筒屏幕显示的名称被正确修改
```



## 3、章筒录入指纹

```
(1)预置条件
    ①所要操作的章筒已被绑定
    ②所要操作的账号是被操作章筒的管理员
    ③账号有权限查看印章列表
(2)操作步骤
    ①进入【通讯录】【印章列表】
    ②点击所要操作的印章，进入印章信息
    ③点击指纹授权，选择授权人员，连接印章
    ④在章筒上录入指纹
    ⑤双击章筒指纹模组，使用新录入的指纹开启印章
(3)预期结果
    ①指纹正确录入
    ②指纹开启印章正常
```



## 4、章筒单据开启使用

```
(1)预置条件
	①小程序单据已申请
(2)操作步骤
    ①进入首页，点击【开启用印】，选择【使用单据】
    ②找到要使用的单据，点击【启动印章】
    ③搜索连接章筒，人脸识别后，选择【现在拍照】
    ④印章开启后使用印章
(3)预期结果
    ①印章可以正确开启
    ②用印次数随着使用相应减少
```



## 5、单据开启使用时长按预警

```
(1)预置条件
    ①小程序单据已申请

(3)预期结果
    ①报警记录正确显示
        1)时间
        2)报警类型
        3)报警章筒名称
```

## 10、章筒红外启动

```
(1)预置条件
    ①用印单据已在cloud端申请，并审核通过
    ②印控台已登录，并进入到用印单据界面
    ③Ocr识别已通过
(2)操作步骤
    ①开启章筒
    ②印控台点击【开启】按钮
    ③将章筒放入印控台【红外范围内】，开始用印
(3)预期结果
    ①印章正确被连接
    ②印章显示次数与单据一致
    ③红外范围之外章筒无法使用
    ④红外范围之内章筒正常使用
```



## 11、章筒删除指纹

```
(1)预置条件
    ①章筒内存有要删除的指纹
    ②所登录账号有删除该章筒指纹权限
(2)操作步骤
    ①进入【通讯录】【印章列表】
    ②点击所要操作的印章，进入印章信息
    ③点击指纹授权，选择【授权用户】，连接印章
    ④指纹删除后，再次使用已删除的指纹尝试打开章筒
(3)预期结果
    ①指纹被正确删除
    ②尝试打开指纹时，章筒提示“未找到相关指纹信息”
```



## 12、解绑章筒

```
(1)预置条件
    ①章筒已被绑定
    ②账号有解绑章筒的权限
    ③要解绑的章筒内指纹已全部被删除
(2)操作步骤
    ①进入小程序【工作台】【印章列表】
    ②长按所要解绑的章筒
    ③点击【删除印章】
    ④在绑定印章界面搜索章筒
(3)预期结果
    ①章筒被正确解绑
    ②再次被所搜时不显示解绑前名称
    ③章筒上屏幕显示“请扫码绑定设备”
```
